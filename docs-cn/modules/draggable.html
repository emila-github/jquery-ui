draggable Module

Defined in: ui-cn\draggable.js:28

可拖拽小部件（Draggable Widget）
允许使用鼠标移动元素。

注释：让被选元素可被鼠标拖拽。如果您不只是拖拽，而是拖拽 & 放置，请查看 jQuery UI 可放置（Droppable）插件，为可拖拽元素提供了一个放置目标。
